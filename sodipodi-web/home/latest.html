11 Veebruar 2004

Sodipodi 0.34 released

The fresh new release of sodipodi is out for your enjoyment. Read Release Notes for the overview of new features. The tarball is immediately available, binary packages will follow very soon.

Packages so far:

  • Source tarball
  • Windows 2000/XP installation package (Unicode version)
  • Windows 9X/ME installation package (ANSI version)
  • Mandrake 9.2 binary and source rpm-s

7 Veebruar 2004

0.34 beta is out

The beta package of the next Sodipodi release is out. Go download it and report back all showstopper bugs.

Available packages

  • Source tarball
  • Windows 2000/XP installation package
  • Windows 9X/ME installation package

You may also try to install Win2000/XP package on 9X/ME system, and report back problems. We would really like to drop multiple builds, but until now have not yet managed to make all-unicode version working on Win9X/ME.
