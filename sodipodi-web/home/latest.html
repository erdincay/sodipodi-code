12 October 2003

Sodipodi Project Status

An October status report for activities in the project has been prepared. Sodipodi got some very positive press on Slashdot, and managed to sneak up to the 19th most active project on SourceForge this past week.

7 October 2003

Sodipodi in SuSE 8.2 Pro

Zaiff Urgulbunger reports that Sodipodi is included with SuSE 8.2 Pro!

6 October 2003

Calling All Patches

A branch in CVS is being set up for integrating patches posted to the SourceForge patch tracker. This branch will be maintained by Ted Gould. Please help us fill up the Sodipodi patch tracker!

30 September 2003

Knoppix 3.2 Includes Sodipodi

Manuel Bastioni reports that Sodipodi is included as an application in the Knoppix Linux distribution, version 3.2.

27 August 2003

Sodipodi 64-bit Port

Mitsuru Oka is initiating work on a port of Sodipodi to IA-64 Linux, using 64-bit hardware resources from the Open Source Development Labs.

UPDATE: The work was a success! Patch has been submitted and accepted.
