22 October 2003

Announcing Sodipodi User Manual Writing Project

The SUM has begun! Cedric Gemy is initiating work on creating a user's manual for Sodipodi, inspired by the Gimp User Manual. Writers and translators are welcome to join. The goal is to provide a complete manual available in many languages.

Join the effort at http://svgfr.net/. So far work has generated a table of contents and incorporated some existing documents. The English version is available here: http://svgfr.net/sum/manual/en/html/index.html


Sodipodi Recommended by EU for Vector Graphics

The EU's Open Source Migration Guidelines lists Sodipodi as the recommended program for working with SVG.


New user contributed artwork for the gallery

Flags for Canada, Lithuania, the UK and Australia have been submitted. Thanks go out to Daniel McRae, Andrius Kurtinaitis, and Michael Grant for helping fill in our flags collection.
