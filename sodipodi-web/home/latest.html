Mail-news gateway

Sodipodi list has now nntp gateway. Just point your newsreader to news.gmane.org and subscribe to gmane.comp.graphics.sodipodi.

16 March 2003

Website updates

There's a new programming tasks page in the development area, for people wishing to work on improving Sodipodi, or interested in seeing development plans for future features. The bugs page has been revised, and the tips and tricks broken out to their own page with a couple new items added.

22 Veebruar 2003

Sodipodi 0.31 has been released

Download it.

UPDATE (24/02/2003): It seems that sodipodi 0.X.1 releases are becoming standard. This time the gradient and pattern fill frontend for gnome-print was not compatible with the rest of sodipodi. And localization works again now ;-)

Read the release notes for the overview of features.

Binaries so far:

  • Mandrake 9.0 binary and source rpm (Xft, lpr) (Lauris)
  • Windows 95/98/2000/XP zipped binary (Read notes) (Lauris)
  • Red Hat Linux 8.0 binary (i386/i686) and source rpms (Dag Wieers)
  • Red Hat Linux 7.3 binary (i386/i686) and source rpms (Lynn Kerby)

Well, the source RPM-s are probably almost identical, but it is nevertheless correct to have one for each customized spec file.
