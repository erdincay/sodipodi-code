0.33beta2 for Windows

The second prerelease beta for windows is out. Go grab it from:
http://www.sodipodi.com/files/Sodipodi_0_33_beta2_Setup.exe
and report back all showstopper bugs.
This version uses Microsoft Layer for Unicode, so feedback from Win9X/ME users is especially welcome.
Reasonably recent Gtk+ runtime is required. Unfortunately the best available package from http://www.dropline.net/gtk/ is broken for Japanese (most probably for Chinese too). We are working on a solution to that problem.
