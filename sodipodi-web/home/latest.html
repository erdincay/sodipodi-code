12 December 2003

New mailing list created

There is now sodipodi-announce mailing list for release (and other relevant) announcements. It was created to give easy monitoring channel for software catalogue owners and distro builders, but is open for subscriptions to everyone. It is moderated list.

Subscriptions page is at: http://lists.sourceforge.net/mailman/listinfo/sodipodi-announce

06 December 2003

Sodipodi 0.33 binaries so far

  • Mandrake 9.2 binary and source rpm (vanilla installation) (Lauris)
  • Windows 2000/XP binary package. It does not work on Win9X because of problems with unicode layer.
  • Windows 95/98/ME binary package. It does work on Win2000/XP as well, although you are limited to ANSI/multibyte charset.
  • Red Hat 9.0 binary and source rpm (Dag)
  • Fedora binary and source rpm (Dag)

Windows binaries require Gtk+ runtime package to be installed. You can download it from Dropline Project or GIMP for windows page. Dropline version has problems with Japanese locale, so if you plan to use it, better install the second package.
