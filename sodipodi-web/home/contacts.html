Getting in touch with Sodipodi people

Mailing lists

There is a single mailing list for sodipodi, where both usage and development is discussed. For subscription and archives, please visit:

http://lists.sourceforge.net/mailman/listinfo/sodipodi-list

There is also very low-volume, low-profile mailing list for announcements. It was created to give easy monitoring channel for software catalogue owners and distro builders, but is open for subscriptions to everyone. It is moderated list.

http://lists.sourceforge.net/mailman/listinfo/sodipodi-announce

Newsgroups

Thanks to GMane sodipodi-list is also mirrored to newsgroup
nntp://news.gmane.org/gmane.comp.graphics.sodipodi

Forums

Sodipodi discussions forums can be found at: http://sourceforge.net/forum/?group_id=4054. If you want new forum to be created, please submit a request to sodipodi-list.

IRC

There is usually #sodipodi channel in GIMPNet IRC server (try irc.gnome.org or irc.gimp.org) to get connected.
