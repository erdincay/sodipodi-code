Getting in touch with Sodipodi people

There is a single mailing list for sodipodi, where both usage and development are discussed. For subscription and archives, please visit:

http://lists.sourceforge.net/mailman/listinfo/sodipodi-list

Also, there is usually #sodipodi channel in GIMPNet IRC server (try irc.gnome.org) to get connected.
