Sodipodi for X11

Starting from Sodipodi 0.29, the list of prerequisites is much shorter than before.

  • Gtk+ 2.0 and Glib 2.0
  • libxml2
  • libart_lgpl
  • libpng
  • libz

The optional libraries are

  • libpopt
  • lignomeprint and libgnomeprintui
  • Xft

If installing libraries from binary rpm packages, you have to install corresponding -devel packages as well. If you do not do, you are fine with binary programs, but cannot compile anything using that library.
