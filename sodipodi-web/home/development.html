Development

Sodipodi welcomes your contributions to help turn it into a great 2D vector drawing tool for the Open Source community.

While many developers work on fixing bugs and creating new features, it is worth strong emphasis that even non-programmers can help make Sodipodi more powerful and successful. You probably already have an idea of something in Sodipodi that you'd like to work on. If not, here are just a few ways you can help:

  • Pick a bug, fix it, and send in a patch
  • Choose a feature you want to see developed, and make it
  • Find a new bug and report it
  • Help answer questions for new Sodipodians on IRC or the mailing list
  • Write an article advocating Sodipodi
  • Author a HOWTO describing a trick or technique you've figured out
  • Set up a gallery to share some examples of things you've made
